{#- How to use the user_banner Everything in this file will be added to the base.html template when the following env vars are set: BROWSE_USER_BANNER_ENABLED=1 BROWSE_USER_BANNER_START_DATE=2022-01-01 BROWSE_USER_BANNER_END_DATE=2022-02-01 These can be set in httpd/conf/ng_flask.conf for use on production. Or they can be set when running flask: BROWSE_USER_BANNER_ENABLED=1 BROWSE_USER_BANNER_START_DATE="2022-01-01" FLASK_APP=app.py flask run The banner will be shown from the first second of BROWSE_USER_BANNER_START_DATE until the last second of BROWSE_USER_BANNER_END_DATE If BROWSE_USER_BANNER_END_DATE is not set, it will be automatically set to one day in the future. This is useful for testing but could cause problems in production. Setting these will also cause an additonal CSS file to be loadded that slides the header away. -#} {# donation banner #}
