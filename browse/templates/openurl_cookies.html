OpenURL settings for arXiv.org

This page allows you to enable or disable OpenURL links on arXiv abstract pages. An OpenURL icon supplied by the linking service you select will appear after to the paper identifier. These links are enabled by setting a cookie which is stored on your machine and sent back to arXiv with each web request. For more details see our OpenURL help page.

Technical details

The OpenURL cookie name is user-OpenURL. Cookies created by this script will be associated with the path {{ path }} on this server and are set to expire in {{ expiry }} years (+{{ expiry }}y). Unless otherwise specified, the URL for the OpenURL link icon is created by appending /sfx.gif to the linking service URL (as specified for OpenURL v0.1). This cookie manipulation script is based on the one developed for OpenURL v0.1 and extended for use with OpenURL v1.0.
